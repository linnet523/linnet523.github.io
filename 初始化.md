更改分辨率（可选，ubuntu26及以上不可用）：
在`~/.config/autostart/n.desktop`中写入
```text
[Desktop Entry]
Type=Application
Name=NVIDIA Custom Scale
Exec=nvidia-settings --assign CurrentMetaMode="DP-0: 1920x1080_144 { ViewPortIn=1728x1080, ViewPortOut=1920x1080+0+0 }"
X-GNOME-Autostart-enabled=true
```

---

重装 nvidia 驱动：
```sh
sudo apt-get remove --purge nvidia*
sudo apt-get autoremove
sudo ubuntu-drivers devices
```

找到 recommended 的驱动名称并复制

`sudo apt install 驱动名称`
